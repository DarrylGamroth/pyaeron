# Documentation

This directory contains user-facing and operational documentation for `pyaeron`.

## Core docs
- `api-reference.md`: public API surface and key types
- `api-contract.md`: behavioral guarantees for lifecycle, callbacks, errors, and ownership
- `troubleshooting.md`: common runtime failures and remediation

## Feature docs
- `advanced.md`: exclusive publications, claims, destinations, counters, images, and CnC
- `callbacks.md`: callback lifetime and exception behavior
- `embedded-driver.md`: in-process media driver usage (`libaeron_driver`)

## Testing and release docs
- `integration.md`: integration harness and CI strategy
- `versioning.md`: versioning and compatibility policy
- `release.md`: release notes for `0.1.0`
